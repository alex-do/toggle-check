toggle-checkbox-radio

CSS only, bootstrap compatible toggle, checkbox and radio buttons.

Download Github

Usage

1. Include toggle-checkbox-radio.css or toggle-checkbox-radio.min.css to your page:

<head>
...
<link href="path/to/toggle-checkbox-radio.css" rel="stylesheet" />
...
</head>

2. Add toggle, checkbox or radio class as examples below:

Toggle

<label class="toggle">
    <input type="checkbox" />
</label>

<label class="toggle">
    <input type="checkbox" /> Toggle with label
</label>

<label class="toggle toggle-sm">
    <input type="checkbox" /> Smaller toggle
</label>

<label class="toggle toggle-lg">
    <input type="checkbox" checked /> Larger toggle default checked
</label>

<label class="toggle toggle-lg">
    <input type="checkbox" checked disabled /> Disabled toggle
</label>


<label class="toggle toggle-primary">
    <input type="checkbox" /> Primary
</label>

<label class="toggle toggle-success">
    <input type="checkbox" checked /> Success
</label>

<label class="toggle toggle-info">
    <input type="checkbox" /> Info
</label>

<label class="toggle toggle-warning">
    <input type="checkbox" /> Warning
</label>

<label class="toggle toggle-danger">
    <input type="checkbox" checked /> Danger
</label>
